1. Clases y Objetos

Concepto

Una Clase es una plantilla o plano para crear objetos. Define las propiedades (características) y métodos (comportamientos) que los objetos de esa clase tendrán. Un Objeto es una instancia real de una clase, con valores específicos para sus propiedades.

2. Propiedades y Métodos

Concepto

Las Propiedades son las variables que definen los atributos o características de un objeto (ej., `Nombre`, `Edad`). Los Métodos son funciones o procedimientos que definen las acciones o comportamientos que un objeto puede realizar (ej., `Saludar()`, `Correr()`).

3. Constructores

Concepto

Un Constructor es un método especial que se llama automáticamente cuando se crea un nuevo objeto de una clase. Su propósito principal es inicializar el estado del objeto, es decir, asignar valores iniciales a sus propiedades. Los constructores tienen el mismo nombre que la clase y no tienen un tipo de retorno.

Simulación: Usando Constructores

Vamos a modificar nuestra clase `Persona` para que el nombre y la edad se asignen directamente al crear el objeto, utilizando un constructor que recibe estos valores como parámetros.

Paso 1: Clase Persona con constructor (simulada):

class Persona
{
    public string Nombre;
    public int Edad;

    // Constructor: Tiene el mismo nombre que la clase
    public Persona(string nombre, int edad)
    {
        Nombre = nombre; // Asigna el parámetro 'nombre' a la propiedad 'Nombre'
        Edad = edad;     // Asigna el parámetro 'edad' a la propiedad 'Edad'
    }
}
